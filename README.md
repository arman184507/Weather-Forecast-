# 🌦️ Weather Forecast

A modern React.js Weather Forecast application that provides real-time weather information for any city using a weather API.

## 🌐 Live Demo

🚀 https://weather-forecast-falcons18.vercel.app/

---

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature
- ☁️ Weather condition
- 📍 Current location weather
- 🔄 Refresh weather data

---

## 🛠️ Built With

- React.js
- JavaScript 
- HTML5
- CSS3
- Weather API

---

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/arman184507/Weather-Forecast-.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```

Open:

```text
http://localhost:3000
```
